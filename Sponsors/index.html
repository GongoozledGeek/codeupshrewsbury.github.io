JetBrains

JetBrains is a software development company whose tools are targeted towards making the lives of other software developers easier.

Their offerings include IDE's for all sorts of languages, including Java, Kotlin, Ruby, Python, PHP, C, Objective-C, C++, C#, Go,[9] JavaScript and SQL. They also offer a great plugin for Visual studio called ReSharper which contains a multitude of tools for VS developers including code analysis, refactoring, navigation, test runner, build runner and more.

We will be raffling one license for one of their products at each CodeUp event!

Redgate

Redgate develops tools for developers and data professionals and maintains community websites such as SQL Server Central and Simple Talk.

Redgate produces specialized database management tools for Microsoft SQL Server, Oracle, MySQL and Microsoft Azure. They also produce advanced developer tools for .NET Framework, such as SmartAssembly and .NET Reflector and have won multiple awards for their products

We will be raffling one license for one of their products at various CodeUp events throughout the year as well as other swag!

Cloudflare

Cloudflare is an infrastructure and website security company, providing content delivery network services, DDoS mitigation, Internet security, and distributed domain name server services.

Trusted by over 20 million Internet properties. Cloudflare provides a scalable, easy-to-use, unified control plane to deliver security, performance, and reliability for on-premises, hybrid, cloud, and SaaS applications.

BCS Shropshire

Shropshire BCS is the professional IT body for the Shropshire area.

Their aim is to help BCS members in Shropshire to have the opportunity:

  • to encourage networking of BCS members in Shropshire and other BCS branches
  • to pass on knowledge of recent developments affecting IT professionals
  • to provide a platform for the discussion of IT issues affecting members
  • to arrange events of interest to BCS members

They host free events to go to where you can sit and watch / listen or get involved with the debates
Their events are organised using Meetup here

As well as offering free access to their events, BCS offer mentoring and help with the organisation of CodeUp and its direction.
